# Kontrola prítomnosti výrobkov – Python aplikácia

Tento projekt slúži na automatizovanú kontrolu počtu a farebného zloženia cumlíkov v balení pomocou spracovania obrazu. Riešenie využíva Python a knižnicu OpenCV a je určené na analýzu snímok alebo priame napojenie na priemyselnú kameru XIMEA.

## 🔧 Inštalácia

Stiahnite si `test_file.exe` z priečinka `dist\` a zložku s testovacími obrázkami z priečinka `images\`.

## ▶️ Spustenie aplikácie (verzia .exe)

Potom spustite z príkazového riadku adresára, v ktorom sa nachádza `test_file.exe:`  

`.\test_file.exe "lokalna cesta k obrazkom"`


Výsledky sa zobrazia priamo v konzole (`OK` / `NOK` pre každé balenie).

## 📄 Dokumentácia

Detailná dokumentácia k softvéru vrátane popisu funkcií, architektúry a technických požiadaviek sa nachádza v súbore `technicka_dokumentacia.pdf` (alebo ako samostatná časť bakalárskej práce).
